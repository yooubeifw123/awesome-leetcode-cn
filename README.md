- [labuladong 的算法小抄](https://github.com/labuladong/fucking-algorithm) - 刷算法全靠套路

- [小浩学算法](https://github.com/geekxh/hello-algorithm) - 针对小白的完整的算法训练流程

- [LeetCode 101](https://github.com/changgyhub/leetcode_101) - 谷歌无人车工程师高分刷题笔记
